# TinyApp Project

TinyApp is a full stack web application built with Node and Express that allows users to shorten long URLs (Ã  la bit.ly).  Registered and logged in users may create, add, update, or delete their short URLs and share these with anyone.  Any user can use the short links but only registered users can create more.

## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session


## Getting Started

- Install all dependencies (using the `npm install` command).
- Run the development web server using the `node express_server.js` command.
